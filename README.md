MiniPLC
=======

An interpreter for the
[MiniPL](https://www.cs.helsinki.fi/u/vihavain/k16/Compilers/project/miniplsyntax_2016.html)
programming language.

Running
-------

    ./gradlew jar
    java -jar build/libs/miniplc.jar helloWorld.mpl
